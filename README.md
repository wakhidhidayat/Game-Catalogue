# Game-Catalogue
An iOS app that displays popular games from Games API https://rawg.io/apidocs

# Features
- Show popular games
- Show game's detail
- Add game to favorites
- Show favorite games
- Show favorite game's detail
- Remove game from favorites
- Author page
